Crudy
=====

iOS NSManangedObject CRUD class category (can sync with a web api)

Just add #import "NSManagedObject+Crudy.h" to your NSManagedObject class and your all set!

Add #import "NSManagedObject+CrudyWebAPI.h" for basic web api syncing. CRUD methods do not sync to server, it just updates from the server state.


WARNING
======

There is currently no implementation of error handling. Use at your own risk!
